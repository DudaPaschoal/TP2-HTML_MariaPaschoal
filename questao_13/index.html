Seletores CSS

Seletor Universal (*)

Todos os elementos da página serão afetados por este seletor. Aplicado a todos os elementos da página. Neste exemplo, definimos a fonte padrão para todos os elementos.

Seletor de Tipo (p, h1, div, etc.)

Seleciona todos os elementos de um tipo específico. Aqui, todos os parágrafos terão a cor azul.

Este é um div.

Seletor de Classe (.)

Este parágrafo tem a classe "destaque". Seleciona elementos com uma determinada classe. O parágrafo com a classe "destaque" terá negrito e cor vermelha.

Seletor de ID (#)

Este título tem o ID "titulo-especial". Seleciona um elemento único com um ID específico. O título com o ID "titulo-especial" terá alinhamento centralizado e um fundo cinza claro.

Seletor de Atributo

Este link possui o atributo href. Seleciona elementos com um atributo específico. Todos os links com o atributo "href" terão a cor verde e sem sublinhado.

Seletor de Descendente

Este parágrafo é descendente de um div com a classe "container". Seleciona elementos que são descendentes de outro elemento. O parágrafo dentro do div com a classe "container" terá estilo itálico e cor blueviolet.
